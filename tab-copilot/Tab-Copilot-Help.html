输入关键词即时过滤，按标题或 URL 搜索，无需鼠标
⌨️

↑↓ 选择标签，Enter 跳转，全程不碰鼠标
📂
域名自动分组
自动按域名归类，一键关闭整组标签
🔄
智能排序
最近活跃 / 字母顺序一键切换，偏好自动跨设备同步

📎
复制 URL
一键复制任意标签的链接到剪贴板
🔒
隐私安全
纯本地运行，不收集任何数据，不发起网络请求
2 安装步骤（手动加载，审核通过前使用）
1
下载压缩包
点击上方 「下载 Tab Copilot」 按钮，下载 tab-copilot-x.x.x-chrome.zip 文件
2
解压文件
解压 zip 到一个文件夹（如：桌面 → tab-copilot）
3
打开扩展管理页
在 Chrome 地址栏输入 chrome://extensions 并回车
4
开启开发者模式
打开页面右上角的 「开发者模式」 开关
5
加载扩展
点击 「加载已解压的扩展程序」，选中第 2 步解压的文件夹
6
固定到状态栏
在 Chrome 工具栏点击 拼图图标（扩展程序）→ 找到 Tab Copilot → 点击 📌 固定，图标常驻工具栏
7
开始使用
点击 Chrome 工具栏的 Tab Copilot 图标即可打开弹窗
温馨提示
  • 手动加载的扩展不会自动更新，商店审核通过后建议去 Chrome Web Store 安装正式版
  • 卸载只需在 chrome://extensions 点击「移除」即可
  • 遇到问题请在 GitHub Issues 反馈：github.com/wellstang/tab-copilot-archive/issues
